# Tank Robot Golem. 

*The robot is controlled by goBLE iOS app.* 

Parts: 
* KKmoon Tracked Robot Smart Car Platform
* Bluno Nano - Arduino Nano Compatible - Bluetooth 4.0
* Power bank 
* Two batteries K18650
* Motor drive controller L298n
* Bread board + Cables
